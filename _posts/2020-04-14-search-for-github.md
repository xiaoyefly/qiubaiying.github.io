# 关于GitHub的搜索技巧以及语法

----------
#### 为什么用github
	因为github提供了很多的功能,比如Web管理界面,评论,组织,点赞、关注、图表,俨然已经是一个基于Github的社交网站,大家围绕着开源项目,进行使用、讨论,贡献等。
	但是很多同学在刚开始用github的时候不知道怎么精确的搜索,可能就是在搜索框搜索,然后一页一页的找,这样非常的耗费时间,本文就是针对这个做了一些关于如何精确搜索的总结
